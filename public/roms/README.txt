Bundled C64 games
=================

Only games that are ours or cleared for redistribution live here.

- guess.prg — "SIDMAKER Guess", an original tiny C64 BASIC game written for
  this project. Released to the public domain.

No commercial or third-party C64 games are distributed in this repo. "Free to
play" is not "free to redistribute": popular homebrew collections explicitly
forbid re-hosting. To play other games, use the "load your own" file picker on
the Games page with a .d64/.prg/.t64/.crt file you legally own, or download
public-domain releases (e.g. from csdb.dk) yourself.
